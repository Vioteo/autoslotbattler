# Многопользовательская веб-игра

Сетевая веб-игра для 2 игроков с системой комнат и автоматическим определением сервера.

## Возможности

- ✅ Создание комнат для игры
- ✅ Подключение к существующим комнатам по ID
- ✅ Автоматическое определение сервера
- ✅ Поддержка до 2 игроков в комнате
- ✅ Real-time коммуникация через WebSocket (Socket.io)
- ✅ Автоматическое переподключение при потере соединения

## Установка

1. Установите зависимости:
```bash
npm install
```

## Запуск

### Простой запуск:
```bash
npm start
```

### Запуск с автоперезагрузкой (для разработки):
```bash
npm run dev
```

Сервер запустится на порту 3000 (или на порту, указанном в переменной окружения PORT).

Откройте браузер и перейдите по адресу: `http://localhost:3000`

## Использование

1. **Создание комнаты:**
   - Нажмите кнопку "Создать комнату"
   - Скопируйте ID комнаты и отправьте его другому игроку
   - Дождитесь подключения второго игрока

2. **Подключение к комнате:**
   - Введите ID комнаты в поле ввода
   - Нажмите "Присоединиться"
   - Игра начнется автоматически, когда оба игрока будут в комнате

## Технологии

- **Node.js** - серверная платформа
- **Express** - веб-сервер
- **Socket.io** - WebSocket библиотека для real-time коммуникации
- **HTML/CSS/JavaScript** - клиентская часть

## Разработка

Для отправки игровых данных используйте событие `gameData`:

```javascript
socket.emit('gameData', {
    roomId: 'ROOM_ID',
    gameData: {
        // Ваши игровые данные
    }
});
```

Для получения данных от другого игрока слушайте событие `gameData`:

```javascript
socket.on('gameData', (data) => {
    // Обработка данных от другого игрока
    console.log(data.gameData);
});
```
